# PRD: 添加减脂餐主题

## 需求概述

在现有项目中添加 5 个减脂餐相关主题及配套菜谱，让用户在选择主题时有更多减脂相关的选项。

## 新增主题

| 主题 ID | 主题名称 | 描述 | Emoji |
|---------|----------|------|-------|
| `boiled-simple` | 水煮无门槛 | 最基础的水煮做法，零厨艺也能做 | 🍲 |
| `harvard-diet` | 哈佛减肥法 | 经典哈佛蔬菜汤，低卡饱腹 | 🥣 |
| `budget-slim` | 穷鬼减肥法 | 省钱又营养，减脂不烧钱 | 💰 |
| `one-white` | 一只白减肥法 | 高蛋白低碳水，增肌减脂 | 🥚 |
| `month-diet` | 窦月减肥法 | 30天循序渐进，稳定掉秤 | 📅 |

## 新增菜谱（每主题 2-3 道）

### 水煮无门槛 (boiled-simple)
1. **水煮鸡胸肉** — 最基础的减脂蛋白质来源
2. **水煮西兰花** — 简单的水煮蔬菜
3. **水煮蛋** — 最便宜的蛋白质

### 哈佛减肥法 (harvard-diet)
1. **哈佛蔬菜汤** — 经典哈佛减肥食谱
2. **番茄洋葱汤** — 低卡饱腹的变体

### 穷鬼减肥法 (budget-slim)
1. **豆腐蔬菜煲** — 廉价高蛋白
2. **土豆鸡蛋沙拉** — 低成本饱腹餐
3. **凉拌黄瓜** — 几乎零成本的开胃菜

### 一只白减肥法 (one-white)
1. **白灼虾** — 高蛋白低脂海鲜
2. **清蒸鱼** — 优质蛋白来源
3. **水煮蛋白** — 极致纯净的蛋白质

### 窦月减肥法 (month-diet)
1. **清炒时蔬** — 温和启动
2. **蒸蛋羹** — 持续控卡
3. **杂粮粥** — 稳定代谢

## 验收标准

- [ ] `src/data/themes.ts` 中新增 5 个主题
- [ ] `src/data/recipes.ts` 中新增配套菜谱（共 12-14 道）
- [ ] 新菜谱的 `themeIds` 正确关联到新主题
- [ ] `npm run build` 成功
- [ ] 应用运行时新主题可正常选择和使用
